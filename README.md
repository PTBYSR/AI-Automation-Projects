# 🤖 AI-Automation-Projects

A curated collection of real-world AI automation systems, production-ready workflows, and intelligent agents. This repository showcases the practical application of Large Language Models (LLMs) and automation platforms to solve complex operational challenges.

---

## 📂 Project Portfolio

### 📨 [Invoice Inbox Automation](file:///c:/Users/paule/OneDrive/Desktop/aap-repo/Invoice%20Inbox%20Automation)
**The Problem:** Manual entry and validation of vendor invoices are slow and prone to error.
**The Solution:** An autonomous agent that monitors Gmail, identifies actual invoices while filtering spam, extracts key financial data via OpenAI, and synchronizes it with a Google Sheets ledger.
- **Tech Stack:** Make.com, OpenAI GPT-4, PDF.co, Google Sheets.

### 📢 [Content Distribution Automation](file:///c:/Users/paule/OneDrive/Desktop/aap-repo/Content%20Distribution%20Automation)
**The Problem:** Creating platform-specific content for social media and newsletters is time-consuming.
**The Solution:** A sophisticated publishing engine that repurposes technical articles into PAS-framed LinkedIn posts, X threads, and newsletters. Includes an AI-powered visual asset generator using Gemini Flash.
- **Tech Stack:** n8n, Gemini 2.5 Flash, GPT-4o, Resend API, Google Drive.

### 🎯 [Lead Generation & Outreach](file:///c:/Users/paule/OneDrive/Desktop/aap-repo/Lead%20Generation%20and%20Outreach%20Automation)
**The Problem:** High-volume outbound sales often lack personalization and data accuracy.
**The Solution:** A high-performance outreach engine that automates lead enrichment, LinkedIn scraping, and email verification. It generates grounded "Founder-to-Founder" copy based on real company context.
- **Tech Stack:** n8n, Apify, Bouncer, GPT-4o, Slack, Airtable.

### 📊 [Reporting & Dashboard Automation](file:///c:/Users/paule/OneDrive/Desktop/aap-repo/Reporting%20and%20Dashboard%20Automation)
**The Problem:** Consolidation of multi-departmental data for year-end reports is a manual bottleneck.
**The Solution:** A one-click reporting workflow that aggregates data across Google Sheets and multiple Airtable bases. Uses Python-validated logic to calculate Sales, Delivery, and HR metrics with 100% accuracy.
- **Tech Stack:** Make.com, Python, Airtable, Google Sheets.

### ☎️ [Voice-Based Customer Support](file:///c:/Users/paule/OneDrive/Desktop/aap-repo/Voice-Based%20Customer%20Support%20Agent)
**The Problem:** High volumes of repetitive support queries strain human agent capacity.
**The Solution:** A RAG-powered voice assistant for RelayPay that resolves routine queries (onboarding, fees, rewards) using an approved knowledge base. Includes a full admin dashboard for escalation tracking.
- **Tech Stack:** Supabase, Vercel, RAG-Architecture, AI Voice Integration.
